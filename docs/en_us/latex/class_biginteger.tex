\doxysubsection{Detailed Description}
It is a big integer class in C++.~\newline
 Since the C++ standard library has no big integer classes,I wrote this IMMATURE \mbox{\hyperlink{class_biginteger}{Biginteger}} class which can complete simple arithmetics including add(+),subtract(-\/),multiply($\ast$),division(/) and so on.~\newline
 Though this class is far behind \mbox{\hyperlink{class_biginteger}{Biginteger}} classes in other high level programming languages such as Java, I still hope to learn something(such as the C++ language,algorithm and \sout{English})from it.~\newline
 All comments and suggestions beneficial to this project are welcome. \begin{DoxyNote}{Note}

\begin{DoxyEnumerate}
\item This class provides an internal attribute(which is a pointer to {\itshape int}) to store the big integer. To increase the efficiency,the big integer will be divided into groups every eight digits. 
\item In {\itshape complexity} of all functions\+:if there are no special notes,{\itshape n} represents the length of {\itshape this,and $\ast$m} represents the length of another one, and {\itshape k=8}. 
\end{DoxyEnumerate}
\end{DoxyNote}

Multiply this integer by another one.~\newline
 {\bfseries{Complexity}}\+: 

The multiplication uses two algorithms


\begin{DoxyEnumerate}
\item If the length of the arrays storing both factors are greater than KARA\+\_\+\+THRESHOLD, the Karatsuba multiplication is used. In this condition,the complexity is around O(n$^\wedge$(1.\+58)),with {\itshape n}represents the length of the bigger factor. 
\item Otherwise,the classic multiplication which behaves alike vertical multiplication you learned in the elementary school is used. In this condition,the complexity is around O(nm / k)
