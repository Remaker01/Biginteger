\doxysubsection{Detailed Description}
It is a big integer class in C++.~\newline
 Since the C++ standard library has no big integer classes,I wrote this IMMATURE \mbox{\hyperlink{class_biginteger}{Biginteger}} class which can complete simple arithmetics including add(+),subtract(-\/),multiply($\ast$),and division(/)(Not ready).~\newline
 Though this class is far behind \mbox{\hyperlink{class_biginteger}{Biginteger}} classes in other High level programming language such as Java,(for instance, it does NOT support negative numbers and bit operations temporarily)I still hope to learn something(such as the C++ language,algorithm and \sout{English})from it.~\newline
 All comments and suggestions beneficial to this project are welcome. \begin{DoxyNote}{Note}

\begin{DoxyEnumerate}
\item This class provides an internal attribute(which is a pointer to {\itshape int}) to store the big integer.\+To increase the efficiency, the big integer will be divided into groups every four digits(to ensure that the mid-\/result of multiply won\textquotesingle{}t exceed the range of {\itshape int}.) 
\item In {\itshape complexity} of all functions, {\itshape n} represents the length of {\itshape this big integer},and {\itshape m} represents the length of another one. 
\end{DoxyEnumerate}
\end{DoxyNote}
