\doxysubsection{详细描述}
It is a big integer class in C++.~\newline
 Since the C++ standard library has no big integer classes,I wrote this IMMATURE \mbox{\hyperlink{class_biginteger}{Biginteger}} class which can complete simple arithmetics including add(+),subtract(-\/),multiply($\ast$),division(/) and so on.~\newline
 Though this class is far behind \mbox{\hyperlink{class_biginteger}{Biginteger}} classes in other high level programming languages such as Java, I still hope to learn something(such as the C++ language,algorithm and \sout{English})from it.~\newline
 All comments and suggestions beneficial to this project are welcome. \begin{DoxyNote}{注解}

\begin{DoxyEnumerate}
\item This class provides an internal attribute(which is a pointer to {\itshape int}) to store the big integer. To increase the efficiency,the big integer will be divided into groups every eight digits. 
\item In {\itshape complexity} of all functions, {\itshape n} represents the length of {\itshape this big integer},and {\itshape m} represents the length of another one. {\itshape k=8} if there are no special notes. 
\end{DoxyEnumerate}
\end{DoxyNote}
